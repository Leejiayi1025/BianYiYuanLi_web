# 编译原理复习系统

> Compiler Principles Review System — 考前冲刺一站式复习

纯前端实现的编译原理复习系统，包含 **17章知识库** 和 **15道期末大题题库**，支持做题、图表渲染、Word导出。

---

## 功能特性

### 知识库（17章）

- 编译程序与解释程序的区别
- 编译过程六阶段
- 词法分析：正则表达式、DFA有穷自动机
- 文法基础、推导与语法树
- FIRST集、FOLLOW集计算
- LL(1)文法判断
- 自下而上分析：移进-归约
- 短语、直接短语、句柄
- 语义分析：符号表管理
- 中间代码：逆波兰式、三地址码、四元式、AST
- 代码优化：基本块与控制流图
- 考前高频题型清单
- 最后速记版

### 题库（15道大题）

- 文法四元组定义与识别
- 正规式与语言描述
- NFA转DFA（子集构造法）
- DFA状态转移与语言识别
- 最左推导与语法树
- FIRST集计算
- FOLLOW集计算
- LL(1)文法判定
- 移进-归约分析
- 短语、直接短语与句柄
- 语义分析与符号表管理
- 逆波兰式与抽象语法树
- 三地址码与四元式
- 基本块划分与流图
- 代码优化：数据流分析

每道题包含：**考察知识点** + **解析** + **完整解答步骤**

### 图表渲染

6道题配有SVG自动渲染图表：

| 题号 | 类型 | 内容 |
|------|------|------|
| Q3 | DFA状态图 | NFA转DFA（A/B/C三状态） |
| Q4 | DFA状态图 | 状态转移（A/B/C/D四状态） |
| Q5 | 语法树 | id + id * id |
| Q10 | 语法树 | E + T * id + F |
| Q12 | 抽象语法树 | (a+b) * (c-d) + e |
| Q14 | 控制流图 | 6个基本块的循环结构 |

### 主题系统

4种Monochrome风格主题，一键切换：

| 主题 | 风格 | 色调 |
|------|------|------|
| Light | 温暖米白 | #F0EDE8 |
| Dark | 深邃灰 | #1A1A1C |
| Warm | 暖调棕灰 | #EAE4DA |
| Cool | 冷调蓝灰 | #E8EAED |

按 `D` 键快速循环切换。

### Word导出

- **导出题目**：15道大题的题目部分，表格和代码块格式工整
- **导出全部**：题目 + 知识点 + 解析 + 解答步骤 + SVG图表
- **导出资料**：17章知识库完整内容

导出格式：`.docx`，表格为真正的Word表格，代码块带灰色背景和左侧竖线。

### 交互功能

- 抽屉式侧边栏，默认展开
- 章节/题目搜索（`Ctrl+K`）
- 学习进度自动追踪
- 面包屑导航
- 字号调整
- 快捷键支持
- 响应式适配

## 快捷键

| 按键 | 功能 |
|------|------|
| `Ctrl+K` / `/` | 聚焦搜索框 |
| `Escape` | 清空搜索 / 关闭面板 |
| `D` | 切换主题（循环4种） |
| `T` | 回到顶部 |
| `←` / `→` | 上一章 / 下一章（知识库） |
| `Enter` | 显示/隐藏解析（题库逐题模式） |

## 技术栈

- **纯前端**：HTML + CSS + JavaScript，无需后端
- **零依赖**：无框架，无构建工具，直接浏览器打开
- **CDN**：docx.js（Word导出）、FileSaver.js（文件下载）
- **SVG**：纯JavaScript生成，无外部图表库

## 使用方式

直接在浏览器中打开 `index.html` 即可。

或使用任意HTTP服务器：

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

## 设计风格

Monochrome（极致黑白摄影风）：

- 颜色严格限制为黑、白、灰（不超过5个灰度层级）
- 标题使用极细字重（100）或极粗字重（900）
- 线条使用0.5px极细线或3px粗线
- 大面积留白（负空间）
- 胶片颗粒感（CSS噪点叠加）
- 克制、冷静、专业
